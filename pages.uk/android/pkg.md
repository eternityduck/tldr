# pkg

> Утиліта менеджеру пакетів для Termux.
> Більше інформації: <https://wiki.termux.com/wiki/Package_Management>.

- Оновити всі встановлені пакети:

`pkg upgrade`

- Встановити пакет:

`pkg install {{package}}`

- Видалити пакет:

`pkg uninstall {{package}}`

- Перевстановити пакет:

`pkg reinstall {{package}}`

- Пошук пакету:

`pkg search {{package}}`
